# Temporary Workspace

This directory is local scratch space for plans, notes, command output, and short-lived experiments.

## Rules

- Do not rely on files here as durable project state.
- Do not commit task plans, generated output, downloaded files, logs, secrets, or large artifacts from this directory.
- Move useful durable notes into `docs/` before they matter to future work.
- Keep only this README and the directory ignore rules tracked.
